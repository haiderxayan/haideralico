# Tech Stack

- **Framework:** Astro 5 (blog starter template configured in `astro.config.mjs`)  
- **Languages & Content:** TypeScript support via `tsconfig.json`, Markdown/MDX content pipelines (`@astrojs/mdx`)  
- **Styling:** Tailwind CSS 3 with the official Astro integration (`@astrojs/tailwind`)  
- **Build & Image Pipeline:** Sharp for image optimization, Astro static site builder  
- **Content & SEO Enhancements:** `@astrojs/rss` for feed generation, `@astrojs/sitemap` for automated sitemap  
- **Tooling:** Node.js/npm scripts (`npm run dev`, `npm run build`, etc.) defined in `package.json`
